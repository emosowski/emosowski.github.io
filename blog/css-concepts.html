CSS Positioning Schemes

February 6, 2016

CSS has three positioning schemes, Static, Relative and Absolute, that allow you to control the layout of a page. Using the position property, you can specify which scheme you would like to implement.

position: static;

STATIC POSITIONING is the default behavior of each element. Each element appears on a new line down the page, regardless of if there is space side by side for two elements.

position: relative;

RELATIVE POSITIONING moves the element from where it would be in static positioning to the top, right, bottom, or left of where it would have been. You can indicate how far to move the element (from where it would have been in static positioning) using the top, bottom, left, or right properties. Relative positioning does not affect the positioning of the elements around it.

position: absolute;

ABSOLUTE POSITIONING takes the element out of normal flow and it does not affect the position of the other elements on the page. The other elements act like the absolute element does not exist. You can use top, bottom, left or right properties to determine where the element should appear in relation to its container.

position: fixed;

FIXED POSITIONING is a type of absolute positioning that places the element in relation to the browser window. When you scroll down the page, a fixed position element stays visible in the exact same place.
